# Implementación de Pila y Cola – Java, JavaScript y Python

Este repositorio contiene la implementación de las estructuras de datos **Pila (LIFO)** y **Cola (FIFO)** en **Java**, **JavaScript** y **Python**.  
El proyecto tiene un enfoque comparativo, educativo y práctico, incorporando pruebas automatizadas y una interfaz básica para visualizar el funcionamiento.

## 🧠 Descripción de los módulos

### ☕ Java (`java-original`)
- Implementación base de Pila y Cola utilizando nodos
- Punto de referencia conceptual del proyecto
- Enfoque didáctico y estructural

### 🌐 JavaScript (`javascript`)
- Implementación en ES6
- Pruebas automatizadas con Jest
- Interfaz simple con HTML y CSS
- Reportes de cobertura de código

### 🐍 Python (`python`)
- Implementación equivalente a Java
- Pruebas automatizadas con Pytest
- Enfoque en claridad, consistencia y validación lógica

---

## 🧪 Pruebas automatizadas

### JavaScript
```bash
cd javascript
npm install
npm test
```

### Python
```bash
cd python
pytest
```
